## 📈 Setup Instructions

To run a real Lighthouse audit:

1. Install Chrome/Chromium:

   ```bash
   brew install chromium

   ```

2. Ensure your application is running at http://localhost:3000

3. Run the audit:
   ```bash
   npm run lighthouse
   ```
